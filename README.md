# BPI Mockups

UI/UX mockups for Bank of the Philippine Islands (BPI) banking applications, created as part of an ideabank initiative to improve digital banking services and customer experience.

## ğŸ“± Mockups Overview

This repository contains 5 comprehensive mockups designed to enhance BPI's digital ecosystem:

### 1. [ATM Status Portal](./atm-status-portal)
**Real-time ATM monitoring system**

A customer-facing portal that provides live status updates on ATM availability across all BPI locations. Helps customers check ATM functionality before traveling and enables staff to monitor the network.

**Key Features:**
- Real-time status tracking (Online, Offline, Under Maintenance)
- Search and filter by location and region
- Service availability information
- Dual authentication (Customer & Admin views)

**Tech Stack:** React, Tailwind CSS, React Router

---

### 2. [Digital Self-Service Issue Logging Kiosk](./digital-self-service-issue-logging-kiosk)
**In-branch issue reporting system**

An interactive kiosk interface for customers to report issues, submit feedback, and log complaints directly from BPI branches without staff assistance.

**Key Features:**
- Touch-optimized kiosk interface
- Issue categorization and priority tagging
- Photo/video attachment support
- Queue number generation

**Tech Stack:** React, Tailwind CSS

---

### 3. [DLU Service Platform](./dlu-service-platform)
**Document and Legal Unit management system**

Internal platform for managing legal documents, compliance tracking, and regulatory requirements across BPI's operations.

**Key Features:**
- Document lifecycle management
- Compliance tracking dashboard
- Role-based access control
- Audit trail and reporting

**Tech Stack:** React, Tailwind CSS

---

### 4. [Outage Complaint System](./outage-complaint-system)
**Service disruption management**

Centralized system for reporting, tracking, and resolving service outages and technical issues across BPI's digital infrastructure.

**Key Features:**
- Incident reporting and categorization
- Status tracking and updates
- Automated notifications
- Analytics and reporting

**Tech Stack:** React, Tailwind CSS

---

### 5. [Visitor Management System](./visitor-management-system)
**Branch visitor tracking and security**

Modern visitor check-in system for BPI branches, enhancing security while streamlining the visitor registration process.

**Key Features:**
- Digital check-in/check-out
- Pre-registration for appointments
- Badge printing integration
- Host notifications

**Tech Stack:** React, Tailwind CSS

---

## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/theresesegui/bpi-mockups.git
   cd bpi-mockups
   ```

2. **Choose a mockup to run**
   ```bash
   cd atm-status-portal
   # or any other mockup folder
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   ```
   http://localhost:5173
   ```
